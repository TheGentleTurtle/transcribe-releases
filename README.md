# Transcribe

A fast, native macOS menu bar app for transcribing YouTube videos.

## Download

Download the latest version from the [Releases](https://github.com/TheGentleTurtle/transcribe-releases/releases) page.

## Features

- Instant YouTube transcriptions from your menu bar
- AI-powered summaries, Q&A, and chat
- Keyboard shortcuts for quick transcription
- Batch transcription support
- Auto-update notifications
- Supports 28+ languages

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel Mac

## Installation

1. Download the `.dmg` from [Releases](https://github.com/TheGentleTurtle/transcribe-releases/releases/latest)
2. Open the DMG and drag Transcribe to Applications
3. Launch from Applications or click the menu bar icon
